Installation
============

```
cd ~
git clone https://github.com/neogenix/dotfiles.git neogenix-dotfiles
cd ~/neogenix-dotfiles
git submodule init
git submodule sync
cd ~
ln -sf neogenix-dotfiles/git/.gitignore ~/.gitignore
```
